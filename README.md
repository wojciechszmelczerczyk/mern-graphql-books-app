## Usage

### Clone repository

```
git clone https://github.com/wojciechszmelczerczyk/mern-graphql-books-app.git
```

### Navigate to project folder

```sh
cd /mern-graphql-books-app
```

### Install dependencies

```
npm i
```

### Run app

#### Run React server

```
npm run start
```

#### Run Apollo GraphQL server

```
npm run server
```

### Env setup

```
# Uri to mongo db
DB_URI=
```
